NMR Predictor Heuristics

This page documents the deliberately simple rules used by the teaching NMR predictor. It is designed to explain trends and spectra layout, not to replace curated or quantum/mechanics-based prediction software.

Scope

Group atoms first, predict shifts second, then render 1D and 2D teaching spectra.

Core Workflow

The predictor is a static browser app, so all chemistry is handled by lightweight JavaScript heuristics.

  • Parse SMILES into a graph of heavy atoms, bonds, aromatic flags, implicit hydrogens, and simple stereochemical markers.
  • Perceive common Kekule aromatic rings so uppercase benzene-like SMILES still behave as aromatic systems.
  • Group atoms by graph/environment keys before calculating ppm values. Grouping is not based on ppm similarity.
  • Assign base shifts from hand-written functional-group rules, then add small substituent/proximity corrections.
  • Expand 1H multiplets using simple n+1 binomial patterns. Component areas are normalized back to the signal integral.
  • Render 1D spectra as area-normalized Gaussian profiles plus centroid sticks. Broad exchangeable peaks use wider lines and therefore lower maxima.

Splitting And 2D Spectra

The couplings are intentionally simplified for teaching multiplicity.

  • Aliphatic 1H splitting uses adjacent carbon-bound hydrogens and binomial n+1 patterns.
  • Aromatic splitting only uses hydrogens on directly adjacent aromatic carbons. Non-equivalent adjacent environments can appear as dd-like patterns.
  • Alkene coupling uses approximate constants: trans ~16 Hz, cis ~10 Hz, unspecified ~12 Hz.
  • HSQC shows one-bond H-C correlations only.
  • COSY shows simple adjacent-proton correlations only.
  • NOESY uses CACTUS 3D coordinates when available, otherwise fallback coordinates. Off-diagonal peaks are proton pairs within 5 A, with relative volume approximately proportional to 1/r^6.

Known Limitations

These limitations are deliberate tradeoffs for a transparent teaching tool.

  • No full spin-system simulation, second-order effects, diastereotopic splitting, or dynamic exchange model.
  • Symmetry/equivalence is graph-environment based and approximate.
  • 13C intensities are shown as approximate carbon counts, not real relaxation/NOE-dependent intensities.
  • Exchangeable OH/NH/SH shifts are broad and solvent dependent; real peaks may move or disappear.
  • CACTUS atom order must match the parsed heavy-atom order before its coordinates are used for NOESY distances.
